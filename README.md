## 如何启动

```sh
pnpm dev
```

打开浏览器扩展管理界面，开启`开发人员模式`，点击`加载解压缩的扩展`，选择 `dist` 目录即可。

> 已支持热重载：修改源码，自动重编译，浏览器自动刷新！

## 发布打包

```sh
pnpm dist
```

打开自己的 Edge 开发者中心，上传 `dist/dist.zip` 即可。
